    檢查 API Key

    確認網頁程式碼中 firebaseConfig 的 apiKey 是否準確無誤？注意大小寫！

  • ❔

    檢查 Google Cloud 金鑰限制

    您的 apiKey 可能設定了限制。進入 Google Cloud「憑證」，編輯該金鑰，確認「應用程式限制」設為「無」，「API 限制」設為「不要限制金鑰」。

    👉 [點此檢查金鑰限制] 🐾

  • ❔

    確認 API 已啟用

    在 Google Cloud 控制台搜尋並啟用「Identity Toolkit API」（Firebase 驗證核心）與「Realtime Database API」。
